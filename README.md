## âœ… Implemented Features

### 1. Create Question with Choices
**POST** `/questions/`

Creates a new quiz question along with its associated choices.

**Request Body Example**
```json
{
  "question_text": "What is FastAPI?",
  "choices": [
    { "choice_text": "Python web framework", "is_correct": true },
    { "choice_text": "Database", "is_correct": false }
  ]
}
```

### 2. Get Question by ID
**GET** `/questions/{question_id}`

Returns a single question by its ID.

### 3. Get Choices for a Question
**GET** `/choices/{question_id}`

Returns all choices associated with a given question.

### 4. Root Endpoint
**GET** `/`
Returns a simple status message confirming that the API is running.
